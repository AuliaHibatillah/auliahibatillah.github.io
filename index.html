Langkah Pembuatan Github Pages dengan GIT

  1. Login ke GitHub
  2. Buat repository baru dengan judul url github page
  3. Buka Git
  4. Melakukan remote terhadap repositori yang ada di Github
  5. Add dan Commit semua file yang dibutuhkan untuk membuat website github page
  6. push file yang sudah di-commit ke dalam repositori GitHub
  7. Cek status repositori
  8. GithubPages sudah dapat diakses
